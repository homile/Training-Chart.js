## React to Chart.js

### 설치하기

기본적으로 Chart.js와 리액트에서 Chart.js를 렌더링하기 위해서는 react-chartjs-2를 설치해야한다.

```
yarn add react-chartjs-2 chart.js
npm install react-chartjs-2 chart.js
```
